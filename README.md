# 🤖 بوت القناة الدينية - Telegram Islamic Content Bot

بوت تليجرام متكامل لنشر محتوى ديني (آيات، أحاديث، أذكار، فيديوهات قرآنية) في قناة تليجرام بشكل تلقائي وذكي.

---

## 📋 المتطلبات

- Python 3.8 أو أحدث
- pip (مدير حزم Python)

---

## 🚀 خطوات التشغيل

### 1️⃣ تحميل الملفات

احفظ الملفات التالية في نفس المجلد:
- `bot.py` - الكود الرئيسي
- `.env` - إعدادات البوت
- `requirements.txt` - المكتبات المطلوبة

### 2️⃣ تثبيت المكتبات

افتح Terminal/Command Prompt في مجلد المشروع واكتب:

```bash
pip install -r requirements.txt
```

### 3️⃣ ضبط الإعدادات

افتح ملف `.env` وتأكد من صحة البيانات:

```env
BOT_TOKEN=7165167784:AAGmUytS5j3lhSoCNbFJ0QaB8CdhThKn1zA
CHANNEL_ID=-1002108494829
ADMIN_ID=5602861672
```

### 4️⃣ تشغيل البوت

```bash
python bot.py
```

## ⚙️ إعدادات البوت في تليجرام

### إنشاء البوت:
1. افتح [@BotFather](https://t.me/BotFather) في تليجرام
2. ابعت `/newbot`
3. اكتب اسم البوت (مثلاً: `قناة الإيمان`)
4. اكتب username (لازم ينتهي بـ `bot`)
5. **احفظ التوكن (Token)**

### إضافة البوت للقناة:
1. افتح إعدادات القناة
2. اضغط "Add Administrator"
3. ابحث عن البوت واضيفه
4. **اعطيه صلاحية "Post Messages"**

### جلب Chat ID:
- افتح القناة في ويب تليجرام
- الرابط هيكون شبه: `https://web.telegram.org/a/#-1002108494829`
- الـ ID هو الرقم بعد `-100` مع إضافة `-100` في البداية

## 🎮 أوامر الأدمن

ابعت أي أمر من دول للبوت في الخاص:

| الأمر | الوظيفة |
|-------|---------|
| `/start` | بدء البوت وعرض الأوامر |
| `/post` | نشر منشور فوري |
| `/stats` | إحصائيات المحتوى |
| `/logs` | آخر 10 منشورات |
| `/reset` | إعادة تعيين المحتوى (ينشر من الأول) |
| `/help` | المساعدة |

## 🧠 مميزات البوت

### ✅ عدم التكرار
- كل آية/حديث/ذكر/فيديو بيتسجل إنه اتنشر
- لما كل المحتوى يخلص، بيرجع يبدأ من الأول
- قاعدة بيانات SQLite محلية

### ✅ الذكاء في الاختيار
- يتناوب بين الأنواع (آيات → أحاديث → أذكار → فيديوهات)
- ما يردش نفس النوع في آخر 2-3 منشورات
- اختيار عشوائي من المحتوى المتاح

### ✅ الجدولة
- ينشر تلقائياً كل 3 ساعات (افتراضي)
- يمكن تغيير الفترة من ملف `.env`
- يمكن ضبط أوقات محددة (مثلاً: الفجر، الظهر، العصر، المغرب)

### ✅ التحكم
- نشر يدوي فوري بأمر `/post`
- متابعة الإحصائيات بأمر `/stats`
- إعادة تعيين المحتوى بأمر `/reset`

## ⚠️ ملاحظات مهمة

### 🔒 الأمان
- **لا تشارك ملف `.env` مع أحد** - يحتوي على التوكن السري
- لو التوكن اتكشف، ارجع لـ [@BotFather](https://t.me/BotFather) واعمل `/revoke`

### 🔄 الاستضافة
لو عاوز البوت يشتغل 24 ساعة:
- **مجاني:** [PythonAnywhere](https://www.pythonanywhere.com) أو [Railway](https://railway.app)
- **مدفوع:** VPS (DigitalOcean, Linode, AWS)
- **سهل:** [Render](https://render.com) أو [Heroku](https://heroku.com)

### 📊 إضافة محتوى
تقدر تضيف آيات/أحاديث/فيديوهات جديدة بتعديل دالة `seed_data()` في ملف `bot.py` أو باستخدام SQL مباشر في قاعدة البيانات.

## 🆘 حل المشاكل

### مشكلة: "Chat not found"
- تأكد إن الـ `CHANNEL_ID` صحيح ويبدأ بـ `-100`
- تأكد إن البوت أدمن في القناة

### مشكلة: "Bot is not a member"
- أضف البوت كأدمن في القناة
- تأكد من صلاحية "Post Messages"

### مشكلة: المحتوى ما بيتنشرش
- تأكد من تشغيل البوت (`python bot.py`)
- افحص ملف `bot.log` للأخطاء
- جرب أمر `/post` يدوياً

### مشكلة: التكرار
- استخدم أمر `/reset` لإعادة تعيين المحتوى
- أو امسح ملف `islamic_content.db` وشغل البوت من جديد
